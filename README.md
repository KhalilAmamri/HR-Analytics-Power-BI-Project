# HR Analytics Power BI Dashboard

## Overview

HR Analytics dashboard that tracks employee attendance, work-from-home trends, and sick leave patterns. Built with Power BI for real-time workforce monitoring.

![Presence Insights Dashboard](Images/Overview%20presense.png)

## Key Metrics

| Metric         | Value  |
| -------------- | ------ |
| Presence Rate  | 91.83% |
| Work From Home | 10.00% |
| Sick Leave     | 1.10%  |

## Files

- **HR-Analytics-Atliq.pbix** - Main Power BI dashboard
- **Attendance-Sheet-2022-2023.xlsx** - Source data (Apr-Jun 2022)

## Dashboard Features

- Employee attendance breakdown by individual
- Presence, WFH, and SL trends over time
- Day-of-week attendance patterns
- Interactive filters for date and employee analysis

## How to Use

1. Open `HR-Analytics-Atliq.pbix` in Power BI Desktop
2. Use filters to drill down by date or employee
3. Review charts for attendance trends and patterns

## Data Period

April - June 2022
